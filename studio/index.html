Current versions of One ROM Studio are unsigned. This means that you may need to bypass your OS's security warnings to install and run it.

  • On Windows, when presented with a "Windows protected your PC", click "More info" > "Run anyway".
  • On Mac, when presented with a "cannot be opened because the developer cannot be verified" warning, go to System Preferences > Security & Privacy, then click "Open Anyway".

Future versions of One ROM Studio will be digitally signed. In the meantime, sha256 checksums for the downloads are provided.

Features
  • Supported platforms: Windows, Mac, Linux (Ubuntu, Debian and Raspberry Pi OS)
  • Supported One ROMs: all variants including Ice/Fire and Pro/USB
  • Program One ROMs with a choice of ROM images
  • Generate and save One ROM firmware files
  • Analyse One ROM devices and firmware files
  • Automatic debug probe and One ROM USB detection
