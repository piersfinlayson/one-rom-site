Windows and Mac versions of One ROM Studio are digitally signed.

However, as a new publisher, Windows Defender Smart Screen may warn that the installer is from an unknown publisher. If this happens:

  • Click "More info"
  • Check the publisher is "Open Source Developer, Piers Finlayson"
  • Click "Run anyway" to proceed

In addition to the Windows and Mac versions being signed, sha256 checksums for all downloads are provided.

Features
  • Supported platforms: Windows, Mac, Linux (Ubuntu, Debian and Raspberry Pi OS)
  • Supported One ROMs: all variants including Ice/Fire and Pro/USB
  • Program One ROMs with a choice of ROM images
  • Generate and save One ROM firmware files
  • Analyse One ROM devices and firmware files
  • Automatic debug probe and One ROM USB detection
