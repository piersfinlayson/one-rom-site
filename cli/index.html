One ROM CLI

Command line One ROM manager for Windows, Mac and Linux

Example Usage 
# Scan for connected One ROMs
$ onerom scan
Scanning ...
found 1 connected device:
  One ROM Fire 24 E - Firmware: 0.6.7 State: Running Serial: 58AD06D6673B9257

# Program a new ROM image
$ onerom program --slot file=c64-kernal.rom,type=2364,cs1=0
Programming device - DO NOT DISCONNECT
Programming complete

# Control the status LED
$ onerom control led flame
The status LED burns like the fires of Mordor

Windows and Mac versions of One ROM CLI are digitally signed.

On Windows and macOS, unzip and extract the One ROM CLI executable and place in a suitable folder in your PATH.

On Linux, install the .deb package as normal. For example, "sudo dpkg -i onerom-cli-x.y.z-1_amd64.deb" (replacing x.y.z with the version number). This will place the "onerom" executable in your PATH.

As a new publisher, on Windows you may get a pop-up entitled "Windows protected your PC" the first time you run the application. If this happens:

  • Click "More info"
  • Check the publisher contains "Open Source Developer, Piers Finlayson"
  • Click "Run anyway" to proceed

In addition to the Windows and Mac versions being signed, sha256 checksums for all downloads are provided.

Features
  • Manage stopped and running One ROMs
  • View and modify the live ROM image being served, instantly
  • Build and program One ROM firmware
  • Perform advanced One ROM management tasks
